# test-skills

Test Claude Code plugin — Unique travel spots finder.

## Install

/plugin marketplace add Milienaa/test-skills
/plugin install test@test-marketplace

## Skills

- **test-hidden-gems** — finds unusual, underrated travel destinations
- **test-trip-planner** — creates day-by-day itineraries for unique spots

## Test prompts

After installing, try:
- "Find me hidden gems in Eastern Europe"
- "Suggest unusual islands few tourists visit"
- "Plan a 5-day trip to somewhere unusual in Central Asia"
